#  Customer Segmentation using KMeans Clustering

This project performs customer segmentation using the KMeans clustering algorithm to identify groups of customers based on purchasing behavior.

An interactive Streamlit application allows real-time prediction and visualization of customer segments.

---

##  Project Overview

Businesses often need to understand different types of customers for targeted marketing strategies.

This system groups customers automatically without labeled data using Unsupervised Machine Learning.

---

##  Methodology

- Feature Selection
- Data Scaling using StandardScaler
- Optimal Cluster Selection using:
  - Elbow Method
  - Silhouette Score
- KMeans Clustering
- Cluster Profiling
- Interactive Visualization

---

##  Tech Stack

- KMeans Clustering
- Streamlit
